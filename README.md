# Roguelike
Console Roguelike 
Написан на C++ с использованием библиотеки ncurses

# Функционал
- Случайная генерация уровней
- Выполнение квестов
- Реализован магазин
- Карта отображается в зависимости от прохождения
- Нападение монстра
- Атака на монстра
- Лифт на другой этаж 
- Можно проходить стэлсом
